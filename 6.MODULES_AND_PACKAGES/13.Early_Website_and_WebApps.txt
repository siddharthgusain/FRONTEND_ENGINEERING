-> Previously Web application were known as "website"
-> Bunch of html pages linked with anchor link tag
-> Each ppage has its own css and javscript
-> Navigating to different page loads the page from server as a seperate network request(GET) along with its other assets.
-> ALSO THESE APPLICATION KNOWN AS MULTI PAGE APPLICATION(MPA);

-> After module bundlers and module support , People started making Websites in terms of 
reusable components and whole UI is rendered with JS 

-> Angular comes into picture as first framework with component driven architecture
-> This leads to website to be renamed as Web Apps and web application started working like native
mobile application.
-> Instead of server side navigation , everything started doing on client-side

-> React became popular as a library

-> These APPLICATION also known As Single Page Application(SPA).
